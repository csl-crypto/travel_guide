# 여행 가이드 프로젝트 작업 규칙

이 폴더는 GitHub Pages에 배포할 정적 웹앱을 보관합니다.

- 기본 파일은 `index.html`, `styles.css`, `app.js` 구조로 유지합니다.
- 빌드 도구 없이 브라우저에서 바로 실행 가능한 형태를 우선합니다.
- 여행 일정 데이터는 `app.js`의 구조화된 상수로 관리합니다.
- 참가자 공유용 화면이므로 모바일 가독성과 조작성을 우선합니다.
- 앱은 원본 캡처 이미지 없이도 동작해야 합니다.
- 원본 캡처 이미지는 참고 자료로만 보고 웹앱에서 직접 참조하지 않습니다.
- 링크는 새 탭에서 열리도록 설정합니다.
